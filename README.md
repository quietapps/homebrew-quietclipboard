# Quiet Clipboard Homebrew tap

Homebrew tap for [Quiet Clipboard](https://github.com/quietapps/QuietClipboard) — a native macOS menu-bar clipboard history manager that silently captures everything you copy and keeps it searchable in a fast, private local library.

## Install

```sh
brew tap quietapps/quietclipboard
brew install --cask quietclipboard
```

## Update

```sh
brew update
brew upgrade --cask quietclipboard
```

## Uninstall

```sh
brew uninstall --cask --zap quietclipboard
brew untap quietapps/quietclipboard
```
